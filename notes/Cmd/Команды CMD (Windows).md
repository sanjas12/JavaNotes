## 📁 Файлы и папки

| Команда | Описание | Пример |
|---|---|---|
| `dir` | Список файлов в папке | `dir C:\Users` |
| `cd` | Сменить папку | `cd C:\Windows` |
| `cd ..` | Перейти на уровень выше | — |
| `md` / `mkdir` | Создать папку | `mkdir MyFolder` |
| `rd` / `rmdir` | Удалить папку | `rmdir /s MyFolder` |
| `copy` | Копировать файл | `copy file.txt D:\` |
| `move` | Переместить файл | `move file.txt D:\` |
| `del` | Удалить файл | `del file.txt` |
| `ren` | Переименовать | `ren old.txt new.txt` |
| `type` | Показать содержимое файла | `type file.txt` |

## 💡 Горячие клавиши и приёмы

- **↑ / ↓** — история команд
- `команда > file.txt` — сохранить вывод в файл
- `команда | clip` — скопировать вывод в буфер обмена
- `cmd1 && cmd2` — выполнить команды последовательно
