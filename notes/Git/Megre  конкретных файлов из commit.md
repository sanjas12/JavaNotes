### Забрать конкретный файл из определённого коммита:

`git restore --source=418dabc04f1d33252871e1a8285ac67a096da9d0 --staged --worktree -- src/logic/logic.py`

Разберём по флагам:

- `--source=<commit>` — из какого коммита берём файл.
    
- `--worktree` — применить в рабочее дерево (чтобы файл реально изменился в папке проекта).
    
- `--staged` — сразу добавить файл в индекс (будет готов к коммиту).
    
- `-- tg-naladka/src/logic/logic.py` — путь до файла (от корня репозитория, без `/` в начале).

 - `--` отделяет хэш коммита от списка файлов (лучше всегда его ставить).
