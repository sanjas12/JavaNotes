Для просмотра истории изменений конкретного файла в Git:

**Базовая история коммитов:**

```bash
git log -- path/to/file.txt
```

**С подробностями (автор, дата, сообщение):**

```bash
git log --oneline -- path/to/file.txt
```

**Показать сами изменения (diff) в каждом коммите:**

```bash
git log -p -- path/to/file.txt
```

**Компактный вид с графиком:**

```bash
git log --oneline --graph -- path/to/file.txt
```

**Посмотреть изменения в конкретном коммите:**

```bash
git show <commit_hash> -- path/to/file.txt
```

**Кто и когда менял каждую строку (blame):**

```bash
git blame path/to/file.txt
```

**Сравнить версию файла между коммитами:**

```bash
git diff <hash1> <hash2> -- path/to/file.txt
```

**Восстановить файл из конкретного коммита:**

```bash
git checkout <commit_hash> -- path/to/file.txt
```

Флаг `--` перед путём к файлу нужен, чтобы Git не перепутал имя файла с именем ветки.
